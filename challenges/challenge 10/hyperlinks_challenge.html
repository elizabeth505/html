Elizabeth Learns HTML

I have learned that when setting a font size, one must use the unit rem instead of px to allow for responsive design. Rem scales the font size in accordance to whatever screen the text is being displayed on.

I have learned that to link external style sheets to a page, one has to put a link tag in the head, with the rel field being stylesheet and the href being the path to the style sheet.

I have learned how to make a form. Forms are used to get and process information from the user. The input tag allows one to create response options in the form of text, radio buttons, passwords, checkboxes, email, and submission. You can also create a dropdown response option, using the select tag with options tags nested underneath.

I have learned a bit of how to use github. Always initialize a repository with a readme, name repositories in lowercase. Use command git clone control v (repository address) to get repo on your local machine. Git add . to start updating changes to the repo, git commit -m"concise message about changes" to commit changes, git push origin -u master to push changes.
